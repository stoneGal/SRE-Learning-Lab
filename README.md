# SRE Learning Lab

Hands-on SRE learning project covering Linux fundamentals, 
monitoring, incident response, and automation.

## Phase 1 — Linux & Monitoring Foundations

### Projects
- **Monitoring Script** — Bash script that checks CPU, disk, 
  and memory every 30 seconds with alerting
- **Disk Full Incident** — Simulated and recovered from a 
  disk full incident, wrote full post-mortem

### Skills Demonstrated
- Linux system administration
- Bash scripting and automation
- Incident response and diagnosis
- Log analysis with grep, awk, sed
- Networking — DNS, TCP, TLS, HTTP
- Service management with systemctl

## Environment
- Ubuntu 22.04 LTS
- Tools: bash, curl, dig, ss, traceroute, systemctl
